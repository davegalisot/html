Insertar elementos en una lista
Vamos a crear la lista de la compra. Para ello dispondremos de unos elementos en la lista e iremos añadiendo los que pongamos en un elemento del tipo input, cuando se pulse el botón añadir. Hay que controlar la visibilidad de este para que sólo se active cuando tenemos algo en el campo input y se desactive cuando agregamos, dejando el input vacío. Tenemos que crearlo mediante la adición de un nodo al elemento UL.
